# Kiviksgraven
A Virtual Reality (VR) experience of the King's Grave near Kivik in southern Sweden.
* [King's Grave (Wiki)](https://en.wikipedia.org/wiki/The_King%27s_Grave)

## Installation
Make sure to have Unity **5.4.2f2**
